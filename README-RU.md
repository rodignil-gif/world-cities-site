# Cestování po městech světa

Этот сайт посвящен путешествиям по городам мира. Здесь вы найдете популярные города и советы.

## Описание

Сайт включает:
- Популярные города
- Путешественные советы

## Технологии

- HTML5
- CSS3 (Bootstrap)
- JavaScript

## Установка

1. Клонируйте репозиторий.
2. Откройте index.html в браузере.
